# vue-input-validate-tester

### 概要  
 入力内容のバリデーションチェック処理を、入力コンポーネントとテスト処理に展開する。  
 テスト処理は複数テストケースを一括テストし結果を表示する。  
 なんちゃってテストツール風で処理させる。  

[![Vue validate tester](http://img.youtube.com/vi/jjKtpDPVTGM/maxresdefault.jpg)](https://youtu.be/jjKtpDPVTGM)

### Set up  
```  
npm install  
npm run dev  
```
